# Тестирование Auth Service

## Unit-тесты

Тестируют бизнес-логику без внешних зависимостей:
- Хеширование паролей
- Создание и проверка JWT токенов
- Регистрация пользователей
- Авторизация
- Валидация данных

Запуск:
```bash
pytest ./tests/unit -v
```

## E2E тесты

Тестируют API endpoints через HTTP запросы. Требуют запущенный сервис.

Запуск:
```bash
docker-compose up -d auth_service
sleep 10
pytest ./tests/e2e -v
```
